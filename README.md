# AI-traffic-light-system
Simulation of AI-controlled traffic signal system using python's pygame

Pythonのpygameライブラリ`を用いてAI制御の交通信号システムをシミュレ`タ`。
Pythonh境g行：Powershellでsystemフォルダのmain.pyをg行します。
Pythonh境がない龊希MainRunning.exeファイルでg行します。

START：プログラムを_始
SUSPEND：一旦停止

r差式：r差式信号を制御
知能式：知能式信号を制御

+：と通行人のl度をUp
－：と通行人のl度をDown
